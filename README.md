# LexiPic - Landing Page de Solución Digital

**LexiPic** es una aplicación móvil innovadora diseñada para promover la autonomía y mejorar la comunicación de personas con afasia o daño cerebral adquirido. Convierte textos complejos en secuencias de pictogramas interactivos y amigables, complementados con opciones de lectura de voz y asistencia inteligente.

Este repositorio forma parte del proyecto del curso de **Diseño de Interfaces de Usuario (IHC)**.

## 🎯 Segmentos Objetivo

1.  **Pacientes (Usuarios Finales)**:
    *   Personas diagnosticadas con afasia de Broca/Wernicke, disartria o daño cerebral adquirido.
    *   Buscan recuperar su independencia para realizar tareas cotidianas (leer guías rápidas, instrucciones o señalizaciones).
    *   Requieren una interfaz minimalista, limpia y libre de distracciones.

2.  **Gestores de Cuidado (Cuidadores, Terapeutas y Familiares)**:
    *   Profesionales de la salud o parientes directos a cargo del cuidado del paciente.
    *   Buscan reducir su carga de asistencia directa y monitorear el progreso del paciente.
    *   Utilizan herramientas de creación de material personalizado y paneles de control clínico.

---

## ✨ Características Principales

*   **Traductor Visual**: Escaneo en tiempo real de textos y conversión automática a secuencias de pictogramas interactivos.
*   **Refuerzo Auditivo**: Herramienta integrada de síntesis de voz (Text-to-Speech) para reproducir el audio de cada pictograma.
*   **Asistencia con Inteligencia Artificial**: Generación automatizada de secuencias visuales personalizadas según las necesidades del paciente.
*   **Feedback Activo**: Sistema interactivo para que el paciente reporte su nivel de comprensión de manera visual y simple.
*   **Tarjeta de Emergencia ("Explicar")**: Tarjeta rápida de identificación para mostrar a extraños en la calle en momentos de bloqueo de habla.
*   **Dashboard Clínico**: Monitoreo detallado del progreso de comprensión de los pacientes e historial de escaneos para terapeutas.
*   **Vinculación Sencilla**: Sincronización instantánea de dispositivos mediante código QR o un código PIN seguro.
*   **Tarjetas Guardadas**: Repositorio offline de conceptos frecuentes listos para ser reproducidos en cualquier instante.

## 🛠️ Tecnologías Utilizadas

- **HTML5**: Estructura de marcado semántica y accesible para lectores de pantalla.
- **CSS3 (Vanilla)**: Diseño adaptable y animaciones sin necesidad de frameworks CSS externos.
- **JavaScript (Vanilla)**: Implementación del menú responsive y la integración de síntesis de voz (Text-to-Speech).
- **Google Fonts & Material Icons**: Tipografía e íconos interactivos premium.
